表单

  • input 属性：
    • maxlength: 最大字符数限制
    • focus
      • true: 启用光标焦点
    • type
      • text: 全键盘
      • number: 数字键盘(0~9)
      • idcard：数字键盘(0~9, X)
      • digit:数字键盘(0~9, .)
    • placeholder: 输入框中默认的提示文字
  • 表单重置

层级规范

  • Popout: 弹出层，内容层和导航层的补充。
    • 承载弹窗通知，操作菜单，菜单，成功或加载中状态的Toast
  • Mask: 蒙层，配合Popout层使用，用于锁定内容层和导航层
  • Navigation: 导航层 位于内容层之上，用户滑动内容层时可保持位置不动。
  • Content: 内容层
